## 微软最强工具集

你听说过 PowerToys 吗？你用过 PowerToys 吗？你可能从来也没听说过，PowerToys 是微软最初发布于 Windows 95 平台的系统增强工具，2019 年，微软宣布开源 PowerToys 实用工具。

在`PowerToys`里集成了多项功能，我最喜欢的功能有颜色选择器、置顶窗口、图像大小调整器等等 ,但只支持 win10、win11

今天，就带你深入了解 PowerToys

## 下载安装

软件网址 [点我进入](https://learn.microsoft.com/zh-cn/windows/powertoys/install)

之后可以通过 github,Microsoft Store 或者 Windows 包管理器安装

我用的是 github 下载完安装软件一路下一步，默认选项即可就行了
